# **What is this project about?**

This project is my attempt to create a game engine. It does not have a name yet, because I don't know if I will be developing it seriously.

---

## **What have I implemented so far and what am I planning?**

At the time of writing this README, I am working on the graphical part of the engine.
It is strictly bound to the Vulkan API, and the core functionality is going to be centered around a render graph (frame graph). If you don't know what that is, you can look it up [here](https://github.com/gfx-rs/gfx/wiki/Frame-graphs#projects).

I will try to make it as flexible and versatile as possible, so users may cut corners where needed but also create their own solutions if necessary.
